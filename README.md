# stock-scoring-system

独立的股票评分系统原型，用 `stock-sdk` 作为数据源，自己维护数据仓库、因子、评分和候选池。

当前阶段是零依赖核心逻辑：

- 数据库 schema SQL：行情快照、日 K、因子、评分、候选池
- 技术因子：动量、波动率、成交额均值、最大回撤
- 综合评分：按因子归一化后加权排序
- 候选池：按资金规模和最大持仓数给出建议资金分配

## 命令

```bash
node --test
```
